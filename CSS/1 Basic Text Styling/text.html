(1)Explain the difference between inline, internal, and external CSS
ANS:: 1_ Inline CSS :: its directly applied in tag, attribute vs Internal CSS :: its applied in head section with style tag vs External CSS :: in this type, make a external file with extension .css and that file link in head section of html file.

(2):: Describe CSS selectors and list the types of selectors e.g., element, class
ANS:: mainly five type of css selectors__
1_Tag Selectors _ ex. p,h1,h3,img,a
2_Class selectors _ its used by multiple html element. class name can be define by .class_name in css ex. .heading, .section
3_ID Selectors _ its used by only one element in html web pages. ex. #heading, #section. ID name define by #IDName
4_Group Selectors _ its used for common style in multiple tag. ex. h1,i,.section
5_Universal Selectors _ it use for applying css style in whole website(we can also say that in tag,class,id,element,attributes).
